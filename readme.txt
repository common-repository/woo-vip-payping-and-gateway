اتصال فروشگاه وکامرس به سرویس پی‌پینگ

== Description ==
با استفاده از این افزونه میتوانید دروازه پرداخت بانکی را و سرویس پیپینگ خود را به فروشگاه اینترنتی متصل کنید.

= ویژگی‌ها =
* امکان تعیین واحد پولی به ریال،تومان، هزار ریال، هزارتومان
* سازگاری با ووکامرس
* همسان‌سازی کدهای تخفیف
* همسان‌سازی آیتم‌ها 
* همسان‌سازی دسته‌بندی‌ها

* این افزونه برای کارکرد صحیح نیازمند به افزونه ووکامرس است. این افزونه شمارا قادر میکند تا تمامی محصولات، کدهای تخفیف و دسته بندیهای حساب پی پینگ خود را به ووکامرس متصل کنید.
* link to the service https://www.payping.ir
* services’ a terms of use and/or privacy policies https://www.payping.ir/terms

== Installation ==

1. پوشه 'woo-vip-payping' را در مسیر '/wp-content/plugins/' آپلود کنید
2. افزونه را از طریق منوی 'افزونه ها' در وردپرس فعال کنید
3. تنظیمات افزونه را می توانید از طریق قسمت تنظیمات ووکامرس / پیکربندی/ درگاه های پرداخت انجام دهید
